# TrustPoll – UI/UX Case Study

## Overview
TrustPoll is a secure, proxy-proof online voting platform designed for institutional elections. The UI/UX focuses on clarity, transparency, and role-based simplicity for both administrators and voters.

## Project Vision
- Enable fair, transparent, and verifiable digital elections.
- Provide simple, high-trust workflows for all users.
- Reduce fraud through multi-layer authentication and structured design.

## Key Design Principles
- Security Through Simplicity
- Trust Through Transparency
- Role-Based Experience Separation

## Problem Space
- Proxy voting and identity misuse
- Lack of real-time monitoring for admins
- Complex voting interfaces causing user errors

## Solution Summary
- Multi-step verification (email, OTP, reCAPTCHA)
- Clear admin dashboards and monitoring tools
- Linear, minimal user flows for error-free voting

## Information Architecture
Admin Flow: Login → 2FA → Dashboard → Create Election → Manage Elections → Manage Voters  
Voter Flow: Signup → Email Verify → Phone OTP → Login → Voting Form → Confirmation
